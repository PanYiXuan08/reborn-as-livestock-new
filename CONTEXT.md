# Reborn as Livestock New Context

## 领域术语

- 再投一胎：中文项目名。动物人生重开模拟器。
- Reborn as Livestock：英文项目名与仓库名来源。
- 狗线家养家庭：首个垂直切片，只验证狗在家养家庭中的一生。
- 成长剧场：局内主玩法界面。玩家通过动作按钮操作动物身体。
- 生命书签：幼年期、成长期、成年期、巅峰期、衰退期、命运日的顶部阶段条。
- 命运偏差：本局不可重抽的出生偏移，会改变操作手感和报告卡。
- 本局小愿望：玩家自选的微目标，报告卡回收。
- 成长记忆：第一次、最强一次、最后一次等行为记录。
- 生平报告：结算页，把玩家操作写回这一世的记忆。

## 当前原型范围

- 只做狗线。
- 只做家养家庭。
- 使用 Godot 4.6.3。
- UI 不使用纯色块。首版加载 `assets/ui_generated` 中的生图资源；若生成资源缺失，回退到 `assets/reference` 中的 UI 设计图参考资源。
- 资源风格：旧档案、黑白绘本、牛皮纸、旧照片、印章。

## 非目标

- 五动物完整内容。
- 正式音效。
- Steam 接入。
- 存档和成就。
- 手柄支持。
